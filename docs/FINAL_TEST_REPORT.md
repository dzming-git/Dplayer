## 🏆 测试框架特性

### 核心功能

1. **测试用例管理**
   - 使用 `@test_case` 装饰器定义测试
   - 支持测试ID、名称、类别、描述、优先级
   - 自动测试发现和注册

2. **断言系统**
   - `assert_equal`: 相等断言
   - `assert_true`: 真值断言
   - `assert_false`: 假值断言
   - `assert_in`: 包含断言
   - `assert_greater`: 大于断言
   - `assert_not_equal`: 不等断言

3. **测试执行**
   - 按类别运行测试
   - 支持全量测试和选择性测试
   - `-k` 参数：错误时继续执行
   - `--all` 参数：运行所有测试

4. **报告生成**
   - 文本格式报告
   - JSON格式报告
   - 测试统计摘要
   - 失败详情和堆栈跟踪
