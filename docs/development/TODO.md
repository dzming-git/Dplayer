### 管理后台日志查看器（开发中 🔨）

**需求描述：**

在管理员后台新增一个「系统日志」标签页，展示 liblog 的四种日志文件内容。

**功能需求：**
1. 表格列：时间、等级/来源、模块、内容（最新在最上面）
2. 支持手机端窄屏（响应式布局）
3. 分页：每页条数可选（10/20/50/100），默认20条
4. 子标签切换日志类型：维护日志 / 运行日志 / 调试日志 / 操作日志
5. 不自动刷新，用户点击「刷新」按钮才刷新；换页不触发刷新（使用已加载数据）
6. 操作日志的等级列显示来源 IP 而非等级

**技术设计：**
- 后端 API：`GET /api/admin/logs?type=maintenance&page=1&limit=20`
  - 读取 `data/logs/{type}.log` 文件
  - 解析日志行为结构化数据
  - 倒序排列（最新在前）
  - 服务端分页
  - 需 root 权限
- 前端：在 Admin.vue 新增 `logs` 标签页
  - 子标签按钮切换日志类型
  - 表格 + 响应式卡片（窄屏）
  - 分页组件 + 每页条数选择
  - 刷新按钮
